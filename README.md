# Vivam Software Services & IT Trainings

Professional company profile and portfolio website built with:
- **Frontend**: React (Craco, Tailwind, Framer Motion)
- **Backend**: FastAPI (Python, MongoDB)

## 🚀 AWS Deployment
We have prepared a step-by-step guide to deploy this application to **AWS Cloud** using **GitHub integration**.

👉 **[Read the Deployment Guide (DEPLOYMENT.md)](./DEPLOYMENT.md)**

## Local Development
### Backend
```bash
cd backend
pip install -r requirements.txt
python server.py
```

### Frontend
```bash
cd frontend
npm install --legacy-peer-deps
npm start
```

## Features
- Dynamic Hero section with glassmorphism.
- Functional Contact Form (saves to MongoDB and sends email).
- Responsive Portfolio and Services sections.
- Integrated Google Maps and SEO-optimized markers.
